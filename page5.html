Yang sering diajukan Oleh Pemula

  1. Root dapat menyebabkan kena virus?  => tidak  Root yang sekarang dipakai adalah Magisk memiliki kemanan yang lebih baik , sisanya tergantung anda  seperti memasang aplikasi sembarangan dan mengizinkan akses  root sembarangan .
  2. Root dapat menyebabkan bootloop dan brick? => bisa jika anda mengedit file sistem dan terjadi kesalahan seperti anda menghapus atau mengedit  file penting pada system maka dapat menyebabkan bootloop dan brick.
  3. Dapatkah saya Mengelock Bootloader kembali? => Anda dapat menlock bootloader anda  tetapi harus mengembembalikan sistem anda terlebih dahulu ke sistem asli / stock rom (rom Pabrikan).
  4. JIka Sudah Root BIsa Kah Saya Melakukan Update Posisi saya Rom Asli pabrik? => Tidak bisa , proses pemasangan update akan gagal atau jika berhasil dapat menyebabkan brick / bootloop
  5. Jika saya sudah root dapatkah saya mengelock Boootloader saya? => Tidak Bisa Root Membutuhkan Bootloader dibuka , jika anda mengunci bootloader dalam keadaan root akan menyebabkan brick.
